# Ebay UI Automation Test

Automation testing project for eBay website using Selenium WebDriver and TestNG.

## 📌 Objective

This project automates the following scenarios:

### Scenario 1 – Apply Filters via Category
- Navigate to Electronics → Cell Phones & Accessories
- Select *Cell Phones & Smartphones*
- Click *All Filters*
- Apply multiple filters (Condition, Price, Item Location)
- Verify filter tags are displayed

### Scenario 2 – Search Product via Search Bar
- Enter search keyword (e.g. MacBook)
- Select category *Computers/Tablets & Networking*
- Click Search
- Verify search results page loads
- Verify first product title contains the search keyword
